# 🚀 Gemini Pro AI - Powerful Chat & Code Assistant

A modern, feature-rich AI chat application powered by Google's Gemini Pro model. This application provides both conversational AI capabilities and advanced code generation features.

## ✨ Features

### 🗨️ Chat Mode
- **Intelligent Conversations**: Powered by Gemini Pro AI model
- **Context Awareness**: Maintains conversation history for better responses
- **Markdown Support**: Rich text formatting with code highlighting
- **Real-time Responses**: Fast and responsive AI interactions
- **Beautiful UI**: Modern, dark theme with smooth animations

### 💻 Code Generation Mode
- **Multi-language Support**: JavaScript, Python, Java, C++, C#, PHP, Ruby, Go, Rust, Swift, Kotlin, TypeScript, HTML/CSS, SQL
- **Framework Integration**: React, Vue.js, Angular, Node.js, Express.js, Django, Flask, Spring Boot, .NET, Laravel
- **Complexity Levels**: Simple, Medium, Complex, Expert Level
- **Test Generation**: Optional unit test generation
- **Production Ready**: Best practices and error handling included
- **Copy to Clipboard**: Easy code copying functionality

### 🎨 User Interface
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Dark Theme**: Easy on the eyes with modern aesthetics
- **Smooth Animations**: Professional feel with CSS animations
- **Intuitive Navigation**: Easy switching between chat and code modes
- **Loading States**: Visual feedback during AI processing

## 📱 Usage

### Chat Mode
1. Click on "Chat Mode" button
2. Type your message in the input field
3. Press Enter or click Send
4. Get intelligent responses from Gemini Pro AI
5. Continue the conversation naturally

**Keyboard Shortcuts:**
- `Enter`: Send message
- `Shift + Enter`: New line

### Code Generation Mode
1. Click on "Code Generation" button
2. Describe what you want to code in detail
3. Select programming language and framework
4. Choose complexity level and test options
5. Click "Generate Code"
6. Copy the generated code to your project

**Code Generation Options:**
- **Language**: Choose from 14+ programming languages
- **Framework**: Select appropriate framework (if applicable)
- **Complexity**: Set code complexity level
- **Tests**: Include unit tests (optional)
